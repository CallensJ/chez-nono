
# CHARTE GRAPHIQUE

Projet : Chez Nono  
Version : 1.1

---

# 1 - INTENTION VISUELLE

L’identité visuelle de Chez Nono repose sur un équilibre entre chaleur humaine, authenticité artisanale et sobriété contemporaine.

La direction artistique doit évoquer :

- la générosité
    
- la convivialité
    
- la cuisine vivante
    
- les matières naturelles
    
- les lieux chaleureux et habités
    

L’ensemble doit rester élégant, lisible et maîtrisé, sans surcharge rustique excessive.

---

# 2 - LOGO

## Structure du logo

Le logo est composé :

- d’un symbole graphique
    
- d’un logotype principal
    
- éventuellement d’une signature secondaire ou baseline
    

## Typographies du logo

### Titre principal

- **Fraunces**
    
- Utilisé pour le nom "Chez Nono"
    
- Style conseillé :
    
    - SemiBold
        
    - Bold
        
    - ou légèrement contrasté selon composition
        

### Signature / accents

- **Caveat**
    
- Utilisé pour :
    
    - slogans
        
    - signatures manuscrites
        
    - mots émotionnels
        
    - détails secondaires
        

## Couleurs du logo

Le logo doit exclusivement utiliser les couleurs de la palette officielle.

Combinaisons recommandées :

- Crème sur fond sombre
    
- Cuivre sur crème
    
- Ambre sur ardoise
    
- Vert foncé + beige chaud
    

Éviter :

- noir pur (#000000)
    
- blanc pur (#FFFFFF)
    
- dégradés excessifs
    
- effets 3D ou ombres fortes

# 4 - TYPOGRAPHIE

## Typographie principale

### Fraunces

Usage :

- titres
    
- hero sections
    
- citations
    
- menus importants
    

Traits :

- élégant
    
- organique
    
- légèrement éditorial
    

---

## Typographie de lecture

### Manrope

Usage :

- paragraphes
    
- navigation
    
- boutons
    
- contenu UI
    

Traits :

- moderne
    
- lisible
    
- neutre
    
- fonctionnel
    

---

## Typographie expressive

### Caveat

Usage limité :

- mots-clés émotionnels
    
- signatures
    
- labels artisanaux
    
- éléments ponctuels
    

Exemples :

- "Convivialité"
    
- "Authenticité"
    
- "Fait maison"
    

Important :  
Caveat ne doit jamais devenir la typographie dominante.

---

# 5 - STYLE VISUEL

## Direction artistique

Le style visuel doit évoquer :

- le bois
    
- les matières naturelles
    
- la lumière chaude
    
- les lieux intimistes
    
- les repas partagés
    

Références visuelles :

- bistrot contemporain
    
- cave à vin chaleureuse
    
- brasserie artisanale moderne
    
- cuisine ouverte vivante
    

---

# 6 - PHOTOGRAPHIE

## Style photo

Privilégier :

- lumière naturelle ou chaude
    
- textures visibles
    
- vapeur, cuisson, matières
    
- humains en action
    
- cadrages proches
    

Éviter :

- photos trop parfaites
    
- banques d’images évidentes
    
- saturation excessive
    
- ambiance clinique

# 7 - UI / INTERFACE

## Boutons

### Primary

- fond ambre (#D97A28)
    
- texte crème (#F7F4EF)
    

### Secondary

- contour cuivre
    
- fond transparent
    

---

## Rayons

- 8px standard
    
- 12px maximum sur cartes larges
    

---

## Espacements

Système basé sur :

- 8
    
- 16
    
- 24
    
- 32
    
- 48
    
- 64
    

---

# 8 - TON VISUEL

L’interface doit être :

- chaleureuse
    
- respirante
    
- structurée
    
- élégante
    
- jamais agressive
    

Le minimalisme recherché est humain, pas technologique.

---

# 9 - APPLICATIONS

Cette charte s’applique à :

- site vitrine
    
- menus digitaux
    
- réseaux sociaux
    
- supports imprimés
    
- contenus promotionnels
    

---

# 10 - PRINCIPE GLOBAL

Chaque élément visuel doit donner l’impression d’avoir été choisi avec intention.

Chez Nono ne cherche pas à impressionner.  
Le lieu cherche à faire rester.
