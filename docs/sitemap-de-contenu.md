
HOME
→ Rôle : captation + orientation + conversion rapide

Sections :
- Promesse principale (identité + ambiance)
- Positionnement du lieu (ce qu’on est / ce qu’on n’est pas)
- Aperçu du menu (séduction visuelle)
- Signature culinaire (ce qui nous rend mémorable)
- Événements (preuve de vie)
- CTA réservation (action immédiate)


MENU
→ Rôle : décision + conversion rationnelle

Sections :
- Introduction courte (philosophie alimentaire)
- Catégories (Sandwiches / Salades / Boissons)
- Chaque item :
  • nom
  • description sensorielle courte
  • prix
- Mise en avant best-sellers


ÉVÉNEMENTS
→ Rôle : storytelling vivant + fidélisation

Sections :
- Introduction (ambiance du lieu)
- Liste d’événements
- Chaque événement :
  • contexte
  • ambiance
  • temporalité
  • appel à venir


À PROPOS
→ Rôle : confiance + identité de marque

Sections :
- Notre histoire (origine, humain, simple)
- Notre concept (ce qu’on propose réellement)
- Notre philosophie (valeurs incarnées, pas slogans)






RÉSERVATION
→ Rôle : conversion pure

Sections :
- Explication simple du process
- Formulaire
- Rassurance (temps, confirmation, disponibilité)


CONTACT
→ Rôle : accessibilité + réassurance

Sections :
- Infos pratiques
- Localisation
- Horaires
- Réseaux sociaux
